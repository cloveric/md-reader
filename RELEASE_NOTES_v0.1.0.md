# md-beader v0.1.0

首个公开版本，聚焦“本地离线、中文友好、开箱可用”的 Markdown 编辑体验。

## Highlights

- 默认 `IR` 所见即所得模式，打开即写
- 三模式切换：`SV / IR / WYSIWYG`
- 工具栏支持常用格式、代码块、列表、链接、表格、预览
- 新建/打开/保存/另存为完整本地文件流
- 核心资源内嵌，离线可用

## Keyboard Shortcuts

- `Ctrl + N` 新建
- `Ctrl + O` 打开
- `Ctrl + S` 保存
- `Ctrl + Shift + S` 另存为

## Download

- `md-beader-v0.1.0-windows-x64.zip`

解压后运行 `md-beader.exe`。
